# ALU Shell Projects

## io_redirections_and_filters

This directory contains shell scripts for I/O redirections and filters.

### Scripts
- 0-hello_world: prints Hello, World
- 1-confused_smiley: displays a confused smiley
- 2-hellofile: displays /etc/passwd
- 3-twofiles: displays /etc/passwd and /etc/hosts
- 4-lastlines: displays last 10 lines of /etc/passwd
- 5-firstlines: displays first 10 lines of /etc/passwd
- 6-third_line: displays third line of iacta
- 7-file: creates a file with a special name
- 8-cwd_state: saves ls -la to a file
- 9-duplicate_last_line: duplicates last line of iacta
- 10-no_more_js: deletes all .js files
- 11-directories: counts directories
- 12-newest_files: displays 10 newest files
- 13-unique: prints unique words
- 14-findthatword: finds root in /etc/passwd
- 15-countthatword: counts bin in /etc/passwd
- 16-whatsnext: shows root and 3 lines after
- 17-hidethisword: hides bin lines
- 18-letteronly: lines starting with a letter
- 19-AZ: replaces A and c
- 20-hiago: removes c and C
- 21-reverse: reverses input
- 22-users_and_homes: displays users and homes
- 23-empty_casks: finds empty files
- 24-gifs: lists gif files
- 25-acrostic: decodes acrostics
- 26-the_biggest_fan: finds top 11 hosts
